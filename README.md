# Parkinson’s Finger Tapping Analysis using AI

## Overview
This project applies computer vision techniques to analyse finger-tapping movements associated with Parkinson’s disease motor assessments.

## Objective
The aim is to extract meaningful movement features such as tapping frequency, amplitude, and fatigue patterns, and explore how these can support clinical scoring systems such as UPDRS.

## Technologies Used
- Python
- YOLOv8 (Ultralytics)
- MediaPipe Hands
- OpenCV
- Roboflow

## Methodology
- Video frame extraction from clinical recordings
- Dataset annotation for supervised learning
- Object detection using YOLOv8
- Hand tracking using MediaPipe
- Feature extraction from movement patterns

## Key Learning
- Importance of high-quality annotated datasets
- Comparison between custom models (YOLO) and pre-trained solutions (MediaPipe)
- Challenges in real-world AI applications
